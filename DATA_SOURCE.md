# 数据源说明

本项目使用 **Baostock** 获取A股数据。Baostock 是证券宝提供的免费证券数据接口。

## 数据接口

| 功能 | API | 说明 |
|------|-----|------|
| 股票列表 | `bs.query_all_stock()` | 全市场股票，本程序过滤为主板（00/60开头） |
| K线数据 | `bs.query_history_k_data_plus()` | 日K线，前复权，含开高低收、成交量、涨跌幅等 |

## 特点

- **免费**：无需注册
- **稳定**：服务器在国内，连接可靠
- **数据准确**：来自证券宝官方
